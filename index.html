How Does Berry Work?
01 Miners begin to work

Miners submit their PoW solution and off-chain data points to the Seer contract.

02 Data Request is Submitted

The user submits a query to Berry Data to incentivize miners to choose this query over other submissions.

03 Wrong Data can be Disputed

Anyone holding Berry governance token (BRY) can dispute the validity of a mined value by paying a dispute fee.

04 Data Feeds are Generated and Distributed

Every 3 minutes, Berry's smart contract selects the best-funded queries and provides a new challenge for miners to solve.
